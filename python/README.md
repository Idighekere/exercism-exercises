## How to Download Exercises

To download an exercise, use the Exercism CLI:

```bash
exercism download --python <exercise-slug>
```

For example:

```bash
exercism download --python hello-world
exercism download --python leap
```

Or to download to a specific directory:

```bash
exercism download --python hello-world --path /home/idighs/dev/02-labs/exercism/python
```
